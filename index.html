JSON

Ressource url:

http://example.com/documentatio/json.php?path=content/doc.txt

Example response:

{
  "file": {
    "title": "Titre de l'article",
    "content": "\ufefftitle: Titre de l'article\n----\nauthor: Auteur de l'article\n----\ndate: 21 d\u00e9cembre 2012\n----\narticle content success"  },
  "meta": {
    "size": "119",
    "response_time": "0.0251"
  }
}
